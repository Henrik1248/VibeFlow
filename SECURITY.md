# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.3.3  | :white_check_mark: |
| < v0.3.3| :x:                |

## Reporting a Vulnerability

We take the security of VibeFlow seriously. If you believe you have found a security vulnerability, please report it to us by opening an issue with the label `security` or contacting the maintainer directly.

**Privacy Guarantee:**
VibeFlow is designed to be local-first.
- Your voice data is processed **locally** using Whisper AI.
- Text refinement is performed **locally** via Ollama (if configured).
- No audio or transcript data is sent to external servers by VibeFlow.
